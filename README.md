# Claude Code Haha

一个宣传 [Claude Code](https://github.com/anthropics/claude-code) 的品牌展示页面，采用 **Chromatic Code** 设计美学——将终端命令行界面升华为视觉诗歌。

## 特性

- 纯 HTML / CSS / JavaScript，零外部依赖
- **星辰粒子系统** — 120 颗闪烁星星 + 粒子连接网络
- **Matrix 雨** — 片假名垂落动画
- **轨道环** — 双旋转同心圆环
- **打字机效果** — 循环输出短语
- **滚动渐入动画** — IntersectionObserver 驱动
- **数字计数器** — 统计数据入场动画
- **全设备适配** — 桌面到手机的响应式布局
- **CRT 质感** — 噪点叠加 + 扫描线

## 设计哲学

> Chromatic Code 将数字智能与模拟工艺融合。命令行不再只是实用工具，而成为发光的画布——每一次按键都在黑暗中留下光的轨迹。

- HTML5 + CSS3 + Canvas API
- 自定义字体：InstrumentSans / JetBrainsMono / GeistMono / Italiana / BricolageGrotesque
- 全站零框架，纯原生实现
